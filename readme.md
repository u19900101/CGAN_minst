## 实现方式：

### 2.普通的神经网络
- 判别器网络为3个隐藏层：输入为图片，输出为图片判别的结果，1为real图片，0为fake
- 生成器网络为4个隐藏层
- 训练网络 将判别器和生成器连起来（固定判别器不训练）：输入为噪音，经生成器后贴上1的标签传入判别器，让fake照片尽可能拟合到real
## 不足：训练1000次和3000次差别不是很大
